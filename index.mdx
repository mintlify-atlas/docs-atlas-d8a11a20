## Get started

<Steps>
  <Step title="Install Grype">
    ```bash
    curl -sSfL https://raw.githubusercontent.com/anchore/grype/main/install.sh | sh -s -- -b /usr/local/bin
    ```
  </Step>
  <Step title="Scan a container image">
    ```bash
    grype alpine:latest
    ```
  </Step>
  <Step title="Review the results">
    Grype displays a table of vulnerabilities with severity, CVE ID, affected package, and fix version. Use `-o json` for machine-readable output.
  </Step>
  <Step title="Integrate with CI/CD">
    Add `--fail-on high` to exit with code `2` if high or critical vulnerabilities are found — perfect for blocking pipelines.
  </Step>
</Steps>
